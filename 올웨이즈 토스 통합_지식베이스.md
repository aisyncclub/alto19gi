### 배송 상태 흐름
```
팀모집완료 → 상품준비중 → 발송중 → 배송중 → 배송완료
```

| 단계 | 설명 |
|------|------|
| 팀모집완료 | 출고 필요 주문. 자정에 자동 이동 |
| 상품준비중 | 엑셀 추출 후 이동 |
| 발송중 | 송장 입력 완료 |
| 배송중 | 택배사 전산 확인 시 **자동 이동** (강제 이동 불가) |
| 배송완료 | 배송완료 후 1영업일 내 자동 이동 |

| 판매자 (불량/오배송) | 판매자 부담 |

- 승인 후 **택배사에 직접 수거 접수** 필요 (자동 안 됨)
- 수거 전 거절: 배송비 환불됨 / 수거 후 거절: 환불 안 됨
- 거절 사유 150자, 수정 불가
- 거절 후 7일 뒤 자동 구매확정

## 15. 토스쇼핑 광고

### 15-1. CPC 성과형 광고 (메인 광고)

| 항목 | 내용 |
|------|------|
| 과금 | 클릭당 과금 |
| 최소 입찰금 | 100원 (10원 단위) |
| 최소 일 예산 | 10,000원 |
| 최소 충전금 | 50,000원 |
| 수수료 혜택 | 클릭 후 7일 내 판매 → 상품판매수수료 0% |
| 부가세 | 입찰금은 VAT 제외, 실제 과금 시 10% 포함 |

### 운영 방식

| 방식 | 설명 |
|------|------|
| **자동 운영** (추천) | 목표 ROAS 입력 → 시스템 자동 입찰 최적화 |
| **수동 운영** | 직접 입찰금 설정 + 타겟(나이/결제내역) 가능 |

**권장 초기 설정**: 자동 운영 + 일 예산 20,000원/상품 + 자동 증액 OFF + 타겟 미설정

### 15-2. 포인트피드 광고 (부스팅)

| 항목 | 내용 |
|------|------|
| 게재비 | 10,000~100,000원 |
| 노출 기간 | 2~10일 (게재비 비례) |
| 과금 | 게재비(고정) + CPC(클릭 시 추가) |
| 문구 | 최대 39자 |
| 심사 | 1~3일 |
| 이미지 | 16:9~9:16, 1280px 이상, JPG/PNG 7MB 이하 |

- 게재 시작 전까지만 취소 가능 (전액 환급)
- 게재 시작 후 취소 **불가**

### 광고비 관리

| 방식 | 설명 |
|------|------|
| 선불(충전) | 5만원 이상 충전, 카드/가상계좌 |
| 판매대금 상계 | 정산 시 광고비 자동 차감 (200만원 한도), **전환 후 선불로 복귀 불가** |

### 성과 지표

| 지표 | 의미 |
|------|------|
| CTR | 클릭률 = 클릭수/노출수 x 100 |
| ROAS | 광고수익률 = 매출/광고비 x 100 |
| 직접 전환 | A 상품 광고 클릭 → A 상품 구매 |
| 간접 전환 | A 상품 광고 클릭 → 같은 셀러 B 상품 구매 |

### 광고 미노출 사유
배송/셀러 페널티, 부당 표시/광고, 썸네일 위반, 토스 추천가 미충족, 재고 없음, 월렛 잔액 부족 → **미노출 기간 광고비 미과금**

|------|------|

# Part 3. 플랫폼 비교 요약

## 올웨이즈 vs 토스쇼핑 핵심 비교

| 항목 | 올웨이즈 | 토스쇼핑 |
|------|---------|---------|
| **배송비** | 전 상품 무료배송 (가격에 포함) | 무료/유료/조건부 선택 |
| **가격 체계** | 개인구매가 + 팀구매가 (10% 초과 인하 필수) | 정상가 + 판매가 + 토스추천가 |
| **수수료** | 카테고리별 차등 (2026.3.3~) | 결제 수수료 + 상품판매수수료 8% |
| **정산 주기** | 구매확정 후 1주일 뒤 금요일 (12~18일) | 구매확정 + 2영업일 |
| **구매확정** | 배송완료 7일 후 자동 | 배송완료 7일 후 자동 |
| **서버 이용료** | 월 매출 100만+ → 49,000원/월 | 없음 |
| **메인 광고** | CPS (판매당 과금) | CPC (클릭당 과금) |
| **광고 수수료 혜택** | 없음 | 클릭 후 7일 내 판매 → 수수료 0% |
| **상품 등급** | S/A/B/C (B이상: 타임특가/쿠폰) | 없음 |
| **페널티 정지** | 30일 10점+ AND 비율 40%+ | 30일 10점+ |
| **송장 규정** | 업로드일 = 집화일 동일 필수 (매우 엄격) | 일반 배송/오늘출발 기한 준수 |
| **복수 상점** | 1사업자 1계정 | 최대 3개 |

# Part 4. 실전 체크리스트

## 올웨이즈 입점~판매 체크리스트

- [ ] 사업자등록증 준비 (800x1200px 이미지)
- [ ] 통신판매업 신고 확인
- [ ] 판매자센터 가입 (비밀번호 특수문자 `!`만 가능)
- [ ] 상품 등록: 팀 구매가 = 최저가, 옵션명 필수, 반품비 왕복
- [ ] 등급 B 이상 유지 → 타임특가/쿠폰 활용
- [ ] 송장 업로드일 = 집화일 동일 (가송장/선송장 절대 금지)
- [ ] 고객 문의 1영업일 내 답변
- [ ] 매주 목요일 데이터 마감 확인

## 토스쇼핑 입점~판매 체크리스트

- [ ] 사업자등록증 + 신분증 준비 (마스킹 불가)
- [ ] 비즈니스 월렛 생성 + 충전 (최소 5만원)
- [ ] 상품 등록: 이미지 600x600px+, 검색 키워드, 필수 고시 입력
- [ ] 토스 추천가 충족 여부 확인
- [ ] 광고: 자동 운영 + 일 예산 20,000원으로 시작
- [ ] 월렛 잔액 수시 확인 (0원 → 광고 중단)
- [ ] 교환/반품 수거는 직접 택배사에 접수
- [ ] 매일 대시보드 점검 (노출/클릭/전환)
