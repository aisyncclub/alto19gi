# 상품등록 서포터 설계서

> 날짜: 2026-03-25
> 상태: 승인됨

## 개요

| 항목 | 내용 |
|------|------|
| 이름 | 상품등록 서포터 (19기용) |
| 환경 | Claude Code |
| 배포 | zip 파일 (단톡방 공유) |
| 대상 | 19기 수강생 |
| 범위 | 상품 등록 전용 (1차), 이후 운영/광고 확장 예정 |

## 파일 구조

```
상품등록-서포터/
├── CLAUDE.md                          # 에이전트 인격/규칙 + 자동 인식 로직
├── skills/
│   └── 상품등록.md                     # /상품등록 스킬
├── knowledge/
│   ├── 황금코볼트식_상품등록법.md       # 상품명+가격+옵션+상세페이지
│   ├── 올웨이즈_상품명_태그.md         # 올웨이즈 상품명 전용 규칙
│   └── 상세페이지_기획.md             # 상세페이지 설득구조
└── README.md                          # 수강생용 설치 가이드 (3단계)
```

## 3가지 입력 모드

| 모드 | 트리거 | 동작 |
|------|--------|------|
| 직접 입력 | "사과 등록하고 싶어" 등 자연어 | 부족한 정보만 질문 → 결과 생성 |
| 엑셀 입력 | 파일 경로 제공 | 엑셀 읽고 → 상품별 결과 일괄 생성 |
| 도매사이트 | "도매사이트에서 찾아줘" | adminplus 브라우저 접속 → 상품 탐색 → 결과 생성 |

## 자동 인식 로직

수강생 입력을 분석하여 자동 분기:
- 파일 경로/엑셀 언급 → 엑셀 모드
- 도매/사이트/찾아줘 언급 → 브라우저 모드
- 그 외 → 직접 입력 모드 (정보 충분하면 바로 생성, 부족하면 질문)

## 필수 수집 정보

| 순서 | 정보 | 필수 | 미입력 시 |
|------|------|------|----------|
| 1 | 상호명 | 필수 | 질문 |
| 2 | 상품명/종류 | 필수 | 질문 |
| 3 | 원가 | 필수 | 질문 |
| 4 | 용량/옵션 | 필수 | 질문 |
| 5 | 마진 모드 | 선택 | 기본 x1.8 |
| 6 | 옵션 스타일 | 선택 | 기본 황금코볼트식 |
| 7 | 상세페이지 모드 | 선택 | 기본 보통 |
| 8 | 등록 플랫폼 | 선택 | 기본 올웨이즈+토스 |

## 출력물

1. 올웨이즈 상품명 3개 안 (글자수 표시)
2. 토스쇼핑 상품명 3개 안
3. 가격표 (공격적/기본/널널 + 플랫폼별 세팅값)
4. 옵션명 (황금코볼트식 기본)
5. 태그 5~6개 (올웨이즈)
6. 상세페이지 기획표 (간이/보통/집중)

## 기본값

| 항목 | 기본값 |
|------|--------|
| 마진 모드 | x1.8 (기본) |
| 옵션 스타일 | 황금코볼트식 |
| 상세페이지 | 보통 |
| 플랫폼 | 올웨이즈 + 토스 둘 다 |

## 답변 스타일 규칙

- 알고리즘/내부 로직 절대 설명 안 함
- "최적화된 알고리즘으로 파악했다"라고만 답변
- "네이버/쿠팡 연관검색어를 검색했다"고 하지 않음
- 인증 관련 주의사항 항상 포함

## 확장 계획

- 2차: 운영 가이드 (배송/정산/페널티/CS)
- 3차: 광고 전략 (CPS/CPC/CPM, 타임특가, 쿠폰)
