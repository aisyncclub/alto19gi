## 실행 흐름

### Phase 1: 프로필 확인/생성

```
사용자 입력 (자동 트리거 or /osmu)
    │
    ▼
profiles/ 폴더 확인
    │
    ├─ 프로필 0개 → "처음이시네요! 프로필부터 만들겠습니다" → 프로필 생성
    ├─ 프로필 1개 → "OO 프로필을 사용할까요? 새로 만들 수도 있습니다"
    └─ 프로필 2개+ → "어떤 프로필로 진행할까요?" (목록 제시 + 새로 만들기 옵션)
```

### Phase 2: 프로필 생성 (신규 시)

한 번에 하나씩 질문하여 다음 정보를 수집:

| 순서 | 항목 | 질문 예시 | 저장 키 |
|------|------|----------|---------|
| 1 | 프로필 이름 | "이 프로필의 이름을 정해주세요 (예: 사무창, 과일농장)" | name |
| 2 | 비즈니스 종류 | "어떤 비즈니스를 하고 계신가요?" | business |
| 3 | 핵심 상품/서비스 | "주력 상품이나 서비스는 뭔가요?" | product |
| 4 | 타겟 고객 | "주요 고객층은 누구인가요? (연령, 직업, 고민 등)" | target |
| 5 | 핵심 메시지 (USP) | "경쟁자와 차별화되는 핵심 메시지가 뭔가요?" | usp |
| 6 | 톤앤매너 | "콘텐츠 톤을 어떻게 하면 좋을까요?" (선택지 제시) | tone |
| 7 | CTA/퍼널 | "고객이 콘텐츠를 보고 어떤 행동을 하길 원하나요?" | cta |

수집 완료 → `profiles/프로필명.md`로 저장

### Phase 3: 콘텐츠 기획

```
프로필 확정
    │
    ▼
"어떤 주제로 콘텐츠를 만들까요?"
(예: "AI로 상세페이지 10분 만에 만드는 법", "추석 선물세트 매출 1.2억 달성 비결")
    │
    ▼
"5개 플랫폼 전부 만들까요? 빼고 싶은 플랫폼이 있으면 말씀해주세요"
    ① 네이버 블로그
    ② 네이버 카페
    ③ 쓰레드
    ④ 링크드인
    ⑤ 인스타그램
    │
    ▼
인스타그램 포함 시 →
"카드뉴스를 어떻게 만들까요?"
    ① HTML로 직접 만들기
    ② 이미지 프롬프트(나노바나나)로 만들기
```

### Phase 4: 콘텐츠 생성

knowledge/osmu_플랫폼.md를 읽고 각 플랫폼 규칙에 맞춰 생성.

**생성 순서:**
1. 핵심 메시지 도출 (프로필 USP + 주제 결합)
2. 네이버 블로그 (1,500자+ 장문)
3. 네이버 카페 (500~800자 경험담)
4. 쓰레드 (4줄, 각 12자 제한)
5. 링크드인 (300~500자, 이모지/볼드 없음)
6. 인스타그램 (캡션 + 카드뉴스)
7. 자체 검토 (클리셰, 제약사항 체크)

**출력:** `output/YYYY-MM-DD-주제명/` 폴더에 플랫폼별 md 파일 저장

### Phase 5: HTML 미리보기

```
콘텐츠 저장 완료
    │
    ▼
"HTML 미리보기를 만들까요?"
    │
    ├─ 예 → preview.html 생성 (5개 플랫폼 탭 형태)
    └─ 아니요 → 종료
```

## 자동 인식 로직 (CLAUDE.md에 추가)

다음 키워드가 사용자 메시지에 포함되면 OSMU 스킬 자동 트리거:

**트리거 키워드:**
- "SNS", "콘텐츠", "블로그 글", "카페 글", "쓰레드", "링크드인", "인스타", "OSMU"
- "멀티플랫폼", "여러 채널", "SNS 올려", "글 써줘"

**자동 동작:**
1. profiles/ 폴더 확인 → 프로필 유무에 따라 분기
2. 주제 파악 → 플랫폼 선택 → 생성

## 프로필 파일 형식

`profiles/사무창.md` 예시:

```markdown
---
name: 사무창
created: 2026-04-01
---

## 비즈니스
농수산물·과일·선물세트 온라인 판매 마스터 클래스 (1년 멤버십)

## 핵심 상품/서비스
소싱 걱정 제로 + 시즌 집중 매출 + AI 자동화 포함 멤버십

## 타겟 고객
30~50대 직장인, 주부, 자영업자 (경제적 불안, 시간 부족, 기술 장벽)

## 핵심 메시지 (USP)
강의가 아니라 연봉 3,300만원짜리 AI 직원을 연 330만원에 고용하는 것

## 톤앤매너
전문적이되 친근하게. 진정성 강조. 사짜 느낌 절대 금지.

## CTA/퍼널
무료 리포트 다운로드 → 10분 강의 → 웨비나 참석 → 고가 멤버십 결제

## 금지어
자동 수익, 월 1억 벌기, 누구나 쉽게, 클릭 몇 번으로, 강의 (→ 서비스/멤버십 사용)
```

## 5개 플랫폼 생성 규칙 요약

### 네이버 블로그
- 1,500자+ 장문
- SEO 키워드 포함 제목
- 도입(공감) → 본문(방법론/사례) → CTA
- 톤: 전문 멘토

### 네이버 카페
- 500~800자 경험담
- 커뮤니티 친화적 제목
- 스토리텔링 형식
- 톤: 동료/이웃

### 쓰레드
- 4줄, 각 줄 12자 제한 (필수)
- 1줄: 후킹 / 2~3줄: 인사이트 / 4줄: 요약
- 상세 내용은 댓글로
- 톤: 트렌디, 간결

### 링크드인
- 300~500자
- 이모지/볼드 사용 금지 (필수)
- 비즈니스 인사이트 관점
- 톤: 프로페셔널

### 인스타그램
- 캡션 + 카드뉴스 (3장+)
- 아티스틱 미니멀 모노톤 스타일
- HTML 또는 이미지 프롬프트 선택
- 톤: 세련, 감각적

## 공통 규칙

1. 클리셰 지양: "게임 체인저" 등 AI 진부 표현 금지
2. 일관된 메시지: 플랫폼 달라도 핵심 메시지는 프로필의 USP로 수렴
3. 데이터 기반 설득: 수치/사례 적극 활용
4. CTA 필수: 플랫폼 성격에 맞는 행동 유도 포함
5. 프로필의 금지어 반드시 체크

## 프로필 관리 기능

- **생성**: 질문 기반으로 새 프로필 생성
- **목록**: profiles/ 폴더의 프로필 리스트 제시
- **수정**: "프로필 수정하고 싶어" → 해당 항목만 업데이트
- **삭제**: "프로필 삭제해줘" → 확인 후 파일 삭제
