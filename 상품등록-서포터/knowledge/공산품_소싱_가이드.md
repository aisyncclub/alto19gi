# 공산품 소싱 가이드 (에이전트용)

> 도매매 위탁판매 중심 전략

## 1. 메인 도매처: 도매매 (domeggook.com)

### 기본 정보
| 항목 | 내용 |
|------|------|
| 사이트 | https://domeggook.com |
| 특징 | **소매(1개 단위) 판매 가능** — 위탁판매 최적 |
| 판매 방식 | 위탁 = 주문 들어오면 도매매에서 구매 → 소비자에게 직배송 |
| 재고 부담 | 없음 (주문 후 구매) |
| 배송 | 도매매 업체가 직접 발송 (소비자 주소 입력) |

### vs 도매국
| 항목 | 도매매 | 도매국 |
|------|--------|--------|
| 최소 수량 | **1개** | 10개 이상 |
| 방식 | 위탁판매 가능 | 사입 전용 |
| 재고 부담 | 없음 | 있음 |
| 초보 적합 | ✅ 최적 | ❌ 리스크 |

## 2. 도매매 상품 찾는 방법

### Step 1: 인기 검색어 확인
- 도매매 메인 페이지 → 인기 검색어 확인
- 인기 검색어 = 현재 셀러들이 활발하게 구매 중 = **지금 팔리는 상품**
- 메인 배너/기획전도 참고

### Step 2: 키워드 검색 + 인기순 정렬
- 원하는 카테고리/키워드로 검색
- **인기 상품순**으로 정렬
- 상위 10개 상품 확인

### Step 3: 업체(공급사) 분석 — 핵심!
상품 클릭 → 업체 정보 확인:

| 확인 항목 | 좋은 신호 | 나쁜 신호 |
|----------|----------|----------|
| 도매 등급 | 골드/실버 이상 | 없음/브론즈 |
| 회사 주소 | 창고/물류센터 주소 | 아파트 주소 ⚠️ |
| 상품 수 | 다양한 상품 보유 | 1~2개만 등록 |
| 후기 | 긍정적 후기 다수 | 배송 지연/불량 후기 |
| 사업 기간 | 1년 이상 | 신규 업체 |

> ⚠️ **아파트 주소 = 도도매(중간 유통) 가능성** → 배송 지연, 품질 불안정 리스크

### Step 4: 위탁 가능 여부 확인
- "위탁판매 가능" 표시 확인
- 1개 단위 구매 가능한지 확인
- 배송비 확인 (무료배송/조건부 무료)

### Step 5: 소량 테스트 주문
- 1~2개 직접 주문하여 품질/배송 확인
- 확인 후 본격 판매 시작

## 3. 시즌 캘린더 (공산품)

### 월별 시즌 상품

| 시즌 | 기간 | 인기 상품 | 비고 |
|------|------|----------|------|
| 봄 | 3~4월 | 황사마스크, 미세먼지 관련, 원예용품, 캠핑용품 | 꽃가루/미세먼지 시즌 |
| 초여름 | 4~5월 | **버킷햇, 자외선차단마스크, 쿨토시/팔토시**, 양산, UV차단 | 🔥 지금 준비! |
| 여름 | 6~8월 | 쿨링용품, 선풍기, 아이스팩, 물놀이, 제습제, 모기퇴치 | 최대 시즌 |
| 가을 | 9~10월 | 김장용품, 보온용품, 핫팩, 텀블러 | 추석 선물 포함 |
| 겨울 | 11~2월 | 핫팩, 방한용품, 기모제품, 전기장판, 크리스마스 | 설 시즌 |
| 신학기 | 2~3월 | 문구류, 가방, 물통, 이름표 | 단기 시즌 |
| 연중 | 상시 | 주방용품, 생활용품, 위생용품, 수납 | 안정 매출 |

### 시즌 상품 준비 타이밍
```
판매 시작 2~3주 전에 등록 + 후기 쌓기
예: 여름용품은 4월 중순부터 등록
예: 겨울용품은 10월부터 등록
```

## 4. 플랫폼별 공산품 전략

### 올웨이즈 공산품 특성
| 항목 | 내용 |
|------|------|
| 주 고객 | 45~65세 여성 |
| 최적 가격대 | 5,000원 ~ 15,000원 |
| 잘 팔리는 것 | 주방용품, 생활용품, 건강/미용, 시즌 필수템 |
| 포인트 | 실용적 + 저렴 + "이 가격에 이게?" 느낌 |
| 수수료 | 카테고리별 10~15% |
| 핵심 | 가격 경쟁력, 충동구매 유도 |

### 토스 공산품 특성
| 항목 | 내용 |
|------|------|
| 주 고객 | 20~40대 |
| 최적 가격대 | 10,000원 ~ 30,000원 |
| 잘 팔리는 것 | 디자인 소품, 주방 인테리어, 선물용, 트렌디한 잡화 |
| 포인트 | 감성적 + 디자인 좋은 + 선물 가능 |
| 수수료 | 판매 8% + 결제 1.6~3.0% |
| 핵심 | **썸네일 글씨 금지**, 미니멀 사진 |

### 가격대별 전략
```
5,000~10,000원   → 올웨이즈 단독 (충동구매)
10,000~20,000원  → 올웨이즈 + 토스 둘 다
20,000~30,000원  → 토스 집중 (디자인/선물)
30,000원 이상    → 토스 단독 (프리미엄)
```

## 5. 공산품 마진 계산

### 기본 공식 (과세 상품)

```
순마진 = 판매가 - 원가 - 수수료 - 택배비 - 부가세

수수료:
  올웨이즈 = 판매가 × 카테고리별 (10~15%)
  토스 = 판매가 × 8% + 판매가 × 결제수수료(1.6~3.0%)

택배비:
  소형 (의류/잡화): 2,500~3,000원
  중형 (주방/생활): 3,000~3,500원
  대형 (가전/가구): 4,000~6,000원

부가세:
  간이과세자: 매출의 약 1~4% (업종별)
  일반과세자: 부가가치의 10%
```

### 마진 배수 (공산품)
| 모드 | 배수 | 용도 |
|------|------|------|
| 공격적 | 원가 x1.8 | 시장 진입, 초기 판매량 확보 |
| 기본 | 원가 x2.0 | 안정적 마진 |
| 널널 | 원가 x2.5 | 프리미엄/독점 상품 |

> 공산품은 식품보다 높은 배수 적용 (CS 리스크 낮고, 유통기한 없음)

### 마진 시뮬레이션 예시

**쿨토시 (원가 2,000원) — 기본 모드 (x2.0)**

| 플랫폼 | 판매가 | 수수료 | 택배비 | 순마진 | 마진율 |
|--------|--------|--------|--------|--------|--------|
| 올웨이즈 | 3,900원 | 468원 (12%) | 2,500원 | -1,068원 ❌ | 적자 |
| → 2개 세트 | 6,800원 | 816원 | 2,500원 | 1,484원 | 21.8% |
| 토스 | 4,900원 | 392원+78원 | 2,500원 | -70원 ❌ | 적자 |
| → 2개 세트 | 7,900원 | 632원+126원 | 2,500원 | 2,642원 | 33.4% |

> ⚠️ **저가 공산품은 반드시 세트/묶음 판매**로 택배비 비중 낮춰야 함!

## 6. 공산품 선정 체크리스트

### 필수 확인
- [ ] 위탁판매 가능한가? (1개 단위 구매)
- [ ] 원가 x2배 후 적정 가격대에 들어오는가?
- [ ] 택배비 비중이 과도하지 않은가? (단가 5,000원 이하면 묶음 판매 필수)
- [ ] 업체 신뢰도 확인 (도매 등급, 주소, 후기)
- [ ] 반품/교환 정책 확인
- [ ] 시즌 상품이면 타이밍 적절한가?

### 피해야 할 상품
| 유형 | 이유 |
|------|------|
| 브랜드 정품 | 지재권/상표권 리스크 |
| 전자기기 | AS/불량 CS 과다 |
| 의류 (사이즈 다양) | 사이즈 교환/반품 CS 과다 |
| 원가 3,000원 이하 단품 | 택배비 비중 과다 → 적자 |
| 식약처 인증 필요 상품 | 인증 없으면 판매 불가 |

### 추천 카테고리 (CS 적고 마진 좋은)
| 카테고리 | 원가 범위 | 장점 |
|---------|----------|------|
| 주방용품 | 3,000~10,000원 | 실용적, 재구매율 높음 |
| 위생/청소 | 2,000~8,000원 | 소모품, 반복 구매 |
| 수납/정리 | 3,000~15,000원 | 디자인 차별화 가능, 토스 적합 |
| 캠핑용품 | 5,000~20,000원 | 시즌 집중, 마진율 높음 |
| 아이디어 소품 | 3,000~10,000원 | SNS 바이럴 가능 |

## 7. 도매매 검색 키워드 DB (시즌별)

