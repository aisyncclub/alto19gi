# 식품 소싱 가이드 (에이전트용)

> 올웨이즈 + 토스쇼핑 식품 판매 전략
> 소싱 → 시즌 매칭 → 마진 계산 → 플랫폼 매칭

---

## 1. 메인 도매처: 강담유통

### 접속 정보
| 항목 | 내용 |
|------|------|
| 파트너 사이트 | https://flinc2333.adminplus.co.kr/partner/login.html |
| 주요 카테고리 | 과일, 채소, 가공식품, 건강식품 |
| 판매 방식 | 위탁판매 (재고 부담 없음) |
| 배송 | 강담유통에서 직접 발송 |
| 결제 | 판매 후 정산 |

### 강담유통 활용 방법
- 검증된 공급처 (강사 직접 거래)
- **엑셀 기반 분석**: 파트너 사이트에서 전체 상품 엑셀 다운로드 → 클로드에 넣으면 자동 분석
- 품질 관리 완료 (CS 리스크 낮음)

### 엑셀 분석 프로세스
```
1. 강담유통 파트너 사이트 접속
2. 전체 상품 엑셀 다운로드
3. 클로드 코드에 파일 드래그 또는 경로 입력
4. "이 엑셀로 지금 시즌에 팔만한 상품 추천해줘"
5. → 시즌 필터링 + 마진 시뮬레이션 + 플랫폼 매칭 자동 실행
```

### 엑셀 파싱 규칙
- python3 + openpyxl 또는 pandas로 읽기
- 컬럼 자동 매칭: 상품명/품명, 원가/매입가/공급가, 카테고리/분류, 용량/규격/중량
- 컬럼명이 다를 수 있으므로 유사 키워드로 자동 탐색
- 파싱 실패 시 사용자에게 어떤 컬럼이 뭔지 질문

| 12월 | 귤, 딸기(초물), 한라봉 | 연말 선물, 크리스마스 |

### 월별 제철 채소/농산물

| 월 | 제철 채소 | 비고 |
|----|----------|------|
| 1~2월 | 시금치, 무, 배추, 당근 | 김장 후 소비 |
| 3~4월 | 냉이, 달래, 두릅, 봄동, 미나리 | 봄나물 시즌 (단기) |
| 5~6월 | 양파, 감자, 마늘, 옥수수 | 햇 농산물 |
| 7~8월 | 옥수수, 고구마, 감자, 토마토 | 여름 채소 |
| 9~10월 | 고구마, 밤, 대추, 잣 | 추석 + 가을 |
| 11~12월 | 배추, 무, 파, 고추 | 김장 시즌 (연매출 최대) |

### 시즌 매출 비중
```
설 시즌 (1~2월):   약 15% — 선물세트 집중
추석 시즌 (8~9월):  약 15% — 선물세트 + 제철 과일
김장 시즌 (11월):   약 10% — 배추, 무, 고춧가루
연말 시즌 (12월):   약 10% — 선물용 과일
비시즌 (그 외):     약 50% — 상시 상품 안정 매출
```

## 3. 플랫폼별 식품 전략

### 올웨이즈 식품 특성
| 항목 | 내용 |
|------|------|
| 주 고객 | 45~65세 여성 |
| 구매 패턴 | 저관여/충동구매, 저렴한 것 선호 |
| 최적 가격대 | 8,000원 ~ 15,000원 |
| 잘 팔리는 것 | 제철 과일, 채소, 가공식품, 건강식품 |
| 안 팔리는 것 | 프리미엄/고급 (30,000원 이상) |
| 수수료 | 신선식품 5.8%, 가공식품 10.6% |
| 핵심 KPI | 매력도 점수 180점 이상 (방문전환율 > 구매전환율 > 재구매율) |

### 토스 식품 특성
| 항목 | 내용 |
|------|------|
| 주 고객 | 20~40대, 디자인 중시 |
| 구매 패턴 | 선물용, 혜택(할인) 추구 |
| 최적 가격대 | 15,000원 ~ 35,000원 |
| 잘 팔리는 것 | 프리미엄 과일, 선물세트, 건강식품 |
| 안 팔리는 것 | 너무 저렴한 것 (1만원 이하) |
| 수수료 | 판매 8% + 결제 등급별 1.6~3.0% |
| 광고 시 | 판매수수료 면제 (결제수수료만) |
| 핵심 | 썸네일 글씨 금지!, 미니멀 디자인 |

### 식품 가격대별 전략
```
8,000~12,000원  → 올웨이즈 집중 (충동구매 유도)
12,000~20,000원 → 올웨이즈 + 토스 둘 다
20,000~35,000원 → 토스 집중 (선물/프리미엄)
35,000원 이상   → 토스 선물세트만
```

## 4. 식품 마진 계산 상세

### 올웨이즈 식품 마진 공식

```
순마진 = 판매가 - 원가 - 수수료 - 택배비

수수료 = 판매가 × 수수료율
  - 신선식품 (과일, 채소, 수산물): 5.8%
  - 가공식품 (건강식품, 간편식 등): 10.6%
  - 기타: 카테고리별 상이 (평균 10%)

택배비 = 3,000 ~ 3,500원 (무게/크기별)
  - 소량 (2kg 이하): 3,000원
  - 중량 (3~5kg): 3,500원
  - 대량 (5kg 초과): 4,000~5,000원

서버비 = 월 33,000원 (등록 상품 수로 나눔, 보통 무시)
```

### 토스 식품 마진 공식

```
순마진 = 판매가 - 원가 - 판매수수료 - 결제수수료 - 택배비

판매수수료 = 판매가 × 8%
  - 단, 광고 판매 시 판매수수료 0% (면제!)

결제수수료 = 판매가 × 등급별
  - 영세: 1.6%
  - 중소1: 2.0%
  - 중소2: 2.5%
  - 일반: 3.0%
  - 초기 셀러 → 영세 기준 1.6%

택배비 = 3,000 ~ 3,500원 (올웨이즈와 동일)
```

### 마진 시뮬레이션 예시

**대저토마토 5kg (원가 15,000원) — 기본 모드 (x1.8)**

| 플랫폼 | 판매가 | 수수료 | 택배비 | 순마진 | 마진율 |
|--------|--------|--------|--------|--------|--------|
| 올웨이즈 | 26,800원 | 1,554원 (5.8%) | 3,500원 | 6,746원 | 25.2% |
| 토스 | 26,800원 | 2,144원 (8%) + 429원 (1.6%) | 3,500원 | 5,727원 | 21.4% |
| 토스(광고) | 26,800원 | 0원 + 429원 (1.6%) | 3,500원 | 7,871원 | 29.4% |

## 5. 식품 상품 선정 체크리스트

### 필수 확인 사항
- [ ] 제철 시즌인가? (제철이면 원가↓ 마진↑)
- [ ] 원가 x1.8배 후 올웨이즈 적정 가격대(8,000~15,000원)에 들어오는가?
- [ ] CS 리스크는 낮은가? (신선식품 → 유통기한/파손 리스크)
- [ ] 택배비 비중이 과도하지 않은가? (원가 대비 택배비 30% 이하)
- [ ] 도매처에서 안정적 공급이 가능한가?
- [ ] 반품/교환 시 손실은 감당 가능한가?

### CS 리스크 등급
| 리스크 | 상품 예시 | 대응 |
|--------|----------|------|
| 🟢 낮음 | 건과류, 가공식품, 양파, 감자 | 상온 배송, 파손 적음 |
| 🟡 보통 | 사과, 배, 감귤, 고구마 | 겨울 동파 주의, 여름 배송 주의 |
| 🔴 높음 | 딸기, 포도, 체리, 수박 | 파손/멍 CS 잦음, 냉장 배송 필요 |

## 6. 식품 상시 추천 (시즌 무관)

### 연중 안정 매출 상품
| 상품 | 원가 범위 | 특징 |
|------|----------|------|
| 양파 | 3,000~6,000원/3kg | CS 적음, 상온, 마진 안정 |
| 감자 | 4,000~7,000원/3kg | CS 적음, 상온, 겨울 동파 주의 |
| 고구마 | 5,000~8,000원/3kg | 인기 높음, 겨울 특히 |
| 건과류 | 8,000~15,000원 | CS 거의 없음, 유통기한 김 |
| 견과류 | 10,000~20,000원 | 선물용 가능, 토스 적합 |
| 건강식품 | 15,000~30,000원 | 토스 프리미엄 적합, 재구매율 높음 |
