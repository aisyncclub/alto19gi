## 과세/면세 구분

마진 계산 시 부가세 처리가 달라지므로 반드시 확인 필요.

| 구분 | 대표 상품 | 부가세 처리 |
|------|----------|------------|
| **면세** | 농산물(과일, 채소), 축산물, 수산물, 임산물 등 미가공 식품 | 부가세 없음 → 마진 그대로 |
| **과세** | 가공식품, 공산품, 생활용품, 뷰티, 패션, 가전 등 | 판매가의 10%를 부가세로 납부 → 마진에서 차감 |

### 면세 vs 과세 마진 차이 예시 (원가 5,000원, 판매가 9,800원 기준)
- **면세**: 순이익 = 9,800 - 5,000 - 택배비 - 수수료
- **과세**: 순이익 = (9,800 - 891부가세) - 5,000 - 택배비 - 수수료 → 부가세만큼 이익 감소

## 마진 계산 시 적용 방법

1. 올웨이즈: 상품 카테고리에 맞는 수수료율 적용 (VAT 별도이므로 실제 부담은 수수료율 × 1.1)
2. 토스쇼핑: 광고 집행 여부 확인 후 적용
3. 카테고리를 모르면 사용자에게 확인하거나, 식품 기본값(신선 5.8% / 가공 10.6%) 적용
4. **과세/면세 반드시 확인** — 면세 상품은 부가세 차감 없이 계산, 과세 상품은 판매가 ÷ 11을 부가세로 차감
