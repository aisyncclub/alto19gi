📖 상품 서포터 사용법
🚀 시작하기
상품 서포터는 올웨이즈/토스쇼핑 판매에 필요한 모든 것을 도와주는 도구입니다.
대시보드(이 페이지) + 클로드 코드(터미널) 두 가지로 사용합니다.
💰 마진 계산기 사용법
1단계: 상단 "💰 마진 계산기" 탭 클릭
2단계: 상품명, 원가, 택배비, 과세구분 입력
3단계: 옵션이 여러 개면 "+ 옵션 추가" 버튼으로 용량/무게별 원가 입력
예) 참외 3kg = 8,000원 / 참외 5kg = 12,000원 / 참외 10kg = 20,000원
4단계: "둘 다 계산" 버튼 클릭 → 올웨이즈 + 토스 마진이 한눈에!
5단계: "💾 상품 저장" 버튼으로 누적 저장
6단계: 아래 "📊 누적 마진 관리"에서 저장된 상품 확인 + 엑셀 다운로드
📊 누적 관리 & 엑셀 다운로드
저장된 상품은 마진 계산기 탭 하단에 누적됩니다.
📥 엑셀 다운로드 버튼으로 전체 상품의 마진을 엑셀로 내보낼 수 있어요.
엑셀에는 옵션별 원가, 판매가, 마진, 마진율이 모두 포함됩니다.
상품 카드를 클릭하면 상세 마진 분석(모드별 비교, 예상 수익)을 볼 수 있어요.
📦 클로드 코드로 상품 등록하기
터미널에서 아래와 같이 입력하면 상품명/가격/옵션/태그/상세페이지가 한번에 생성됩니다:
상호명: 황금상회
상품: 성주 참외 3kg/5kg
원가: 3kg 8,000원 / 5kg 12,000원

이 상품 등록해줘
마진 계산만 하고 싶으면: "참외 원가 8000원 마진 계산해줘"
엑셀 대량 등록: 엑셀 파일 경로와 함께 "이 엑셀로 등록해줘"
🔍 상품 찾기 & 시즌 캘린더
상품 찾기 탭: 도매 엑셀을 업로드하면 시즌 적합도, 마진율, 플랫폼 매칭까지 자동 분석
시즌 캘린더 탭: 월별 제철 과일/채소 + 공산품 시즌 키워드 확인
바로 등록 탭: 복사해서 클로드 코드에 붙여넣으면 바로 등록 시작
💡 꿀팁
• 마진율 30% 이상이면 안정적, 15% 미만이면 가격 조정 필요
• 올웨이즈는 타임특가 20% + CPS가 광고비 무료라 가장 유리한 경우가 많아요
• 토스쇼핑은 광고 집행 시 운영수수료 8% 면제 → 수수료 2%만!
• 옵션 원가가 다르면 반드시 옵션별로 입력해야 정확한 마진을 알 수 있어요
• 저장된 상품은 엑셀로 다운받아서 스프레드시트에서 관리하세요
