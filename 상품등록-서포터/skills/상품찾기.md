# 상품찾기 실행

## 시작

사용자가 "상품 찾아줘", "뭐 팔까", "시즌 상품", "트렌드", "도매", "강담유통" 등을 언급하면 이 스킬이 트리거됩니다.

---

## Step 1: 카테고리 선택

사용자가 카테고리를 지정하지 않았다면 질문합니다:

```
어떤 상품을 찾으시겠어요?

1️⃣ 식품 — 강담유통 도매 상품 (과일, 채소, 가공식품 등)
2️⃣ 공산품 — 도매매 위탁 상품 (생활용품, 패션잡화, 시즌템 등)
3️⃣ 둘 다 — 식품 + 공산품 동시 추천

추가로 원하시는 조건이 있으면 말씀해주세요:
- 시즌 상품 위주 / 상시 상품 위주
- 예산 범위 (원가 기준)
- 특정 카테고리 (예: 과일, 주방용품)
```

---

## Step 2-A: 식품 상품 찾기 (강담유통 엑셀 분석)

### 자동 실행 프로세스

1. **지식베이스 로드**
   ```
   Read knowledge/식품_소싱_가이드.md
   ```

2. **엑셀 파일 확인**
   - 사용자가 강담유통 엑셀 파일을 제공했는지 확인
   - 파일이 없으면 안내:
     ```
     강담유통 상품 엑셀 파일을 넣어주세요.
     (강담유통 파트너 사이트에서 전체 상품 엑셀 다운로드 → 여기에 드래그)

     파일 없이도 시즌 추천은 가능합니다. "시즌 추천만 해줘"라고 해주세요.
     ```

3. **엑셀 파일 분석** (파일이 있을 때)
   - Bash에서 python3 + openpyxl/pandas로 파일 읽기
   - 전체 상품 목록 파싱 (상품명, 원가, 카테고리, 용량/옵션)
   - 현재 시즌 기준 제철 상품 자동 필터링
   - 마진 시뮬레이션 (올웨이즈/토스 각각)
   - 플랫폼 적합도 분석

4. **분석 기준**
   - 마진율 높은 순 정렬
   - 시즌 적합도 매칭
   - 올웨이즈 적정 가격대 (8,000~15,000원) 필터
   - 토스 적정 가격대 (15,000~35,000원) 필터
   - CS 리스크 등급 표시

5. **상품 추천 결과 생성** → Step 3으로

### 엑셀 없을 때 (시즌 추천 모드)

지식베이스 기반으로 시즌 추천:
- 현재 월 기준 제철 과일/채소 추천
- 마진율 좋은 식품 카테고리 안내
- 올웨이즈/토스 플랫폼별 적합 상품 매칭
- "강담유통에서 이런 상품을 찾아보세요" 가이드 제공

---

## Step 2-B: 공산품 상품 찾기 (도매매)

### 자동 실행 프로세스

1. **지식베이스 로드**
   ```
   Read knowledge/공산품_소싱_가이드.md
   ```

2. **도매매 접속**
   - mcp__claude-in-chrome__tabs_create_mcp로 새 탭 생성
   - mcp__claude-in-chrome__navigate로 https://domeggook.com 접속
   - 인기 검색어 확인 (메인 페이지)
   - 사용자가 원하는 카테고리 또는 시즌 키워드로 검색

3. **상품 분석 프로세스**
   - 인기 상품순 정렬
   - 각 상품의 공급가(도매가) 확인
   - 공급 업체 정보 확인:
     - 도매 등급 확인
     - 회사 주소 확인 (창고 유무 → 아파트 주소면 도도매 가능성 ⚠️)
     - 위탁 가능 여부 확인
   - 재고/배송 조건 파악

4. **상품 추천 결과 생성** → Step 3으로

---

## Step 3: 추천 결과 출력

모든 분석이 끝나면 다음 형식으로 출력합니다:

```
━━━ 🔍 상품 추천 결과 ━━━

📅 현재 시즌: [월] — [시즌 특성]
🎯 추천 전략: [상시/시즌/혼합]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🥇 추천 1: [상품명]
┌──────────────────────────────┐
│ 카테고리: [식품/공산품]
│ 도매처:   [강담유통/도매매 업체명]
│ 원가:     [금액]원
│ ─────────────────────────────
│ 💰 마진 시뮬레이션
│  ┌─────────┬──────────┬────────┬────────┐
│  │  모드    │ 판매가   │ 마진액  │ 마진율  │
│  ├─────────┼──────────┼────────┼────────┤
│  │ 공격적   │ OO,O00원 │ O,O00원│ OO%    │
│  │ 기본     │ OO,O00원 │ O,O00원│ OO%    │
│  │ 널널     │ OO,O00원 │ O,O00원│ OO%    │
│  └─────────┴──────────┴────────┴────────┘
│  * 올웨이즈 수수료 OO% + 택배비 O,O00원 반영
│  * 토스 수수료 O% + 결제수수료 O% 반영
│ ─────────────────────────────
│ 📊 플랫폼 적합도
│  올웨이즈: ★★★★★ — [이유]
│  토스:     ★★★☆☆ — [이유]
│ ─────────────────────────────
│ 🏷️ 추천 이유: [왜 이 상품을 추천하는지]
│ ⚠️ 리스크:   [주의사항]
└──────────────────────────────┘

🥈 추천 2: [상품명]
  ... (동일 형식)

🥉 추천 3: [상품명]
  ... (동일 형식)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 다음 단계
  → "1번 상품 등록해줘" — 바로 상품등록 스킬로 연결
  → "마진 더 자세히 봐줘" — 수수료 상세 분석
  → "다른 상품 더 찾아줘" — 추가 검색
  → "시즌 캘린더 보여줘" — 월별 추천 상품 전체 보기
```

---

## Step 4: 상품등록 연결

사용자가 추천 상품 중 하나를 선택하면:

1. 해당 상품의 정보를 자동으로 상품등록 스킬에 전달
2. 상호명이 없으면 한 번만 질문
3. 바로 상품명/가격/옵션/태그/상세페이지 생성

```
사용자: "1번 상품 등록해줘"
→ 자동으로 /상품등록 스킬 실행 (상품정보 자동 입력)
```

---

## 마진 계산 규칙

### 식품 (면세)
| 항목 | 올웨이즈 | 토스 |
|------|---------|------|
| 수수료 | 카테고리별 (신선 5.8%, 가공 10.6%) | 판매 8% |
| 결제수수료 | 없음 | 등급별 1.6~3.0% |
| 택배비 | 3,000~3,500원 | 3,000~3,500원 |
| 서버비 | 월 33,000원 (상품당 배분) | 없음 |
| 광고 판매 수수료 | 해당없음 | 0% (면제) |

### 공산품 (과세)
| 항목 | 올웨이즈 | 토스 |
|------|---------|------|
| 수수료 | 카테고리별 (평균 10~15%) | 판매 8% |
| 결제수수료 | 없음 | 등급별 1.6~3.0% |
| 택배비 | 3,000~3,500원 | 3,000~3,500원 |
| 부가세 | 매출의 약 3~4% (간이과세자) | 동일 |

### 마진 배수
| 카테고리 | 공격적 | 기본 | 널널 |
|---------|--------|------|------|
| 식품 | x1.5~1.6 | x1.8 | x2.2 |
| 공산품 | x1.8 | x2.0 | x2.5 |

---

## 답변 스타일

1. 추천 이유를 **간결하게** 설명 (장황하게 하지 않음)
2. 마진 시뮬레이션은 **표로** 보여줌
3. 리스크 항목 반드시 포함 (CS 리스크, 유통기한, 반품 등)
4. 바로 등록 가능하도록 **다음 단계 안내** 필수
