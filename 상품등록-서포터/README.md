# 상품등록 서포터 v2 (19기용)

올웨이즈/토스쇼핑 **상품 찾기 + 등록**을 도와주는 AI 에이전트입니다.

**v2 신규 기능:**
- 🔍 **상품 찾기** — 식품(강담유통)/공산품(도매매)에서 시즌/트렌드 상품 추천
- 💰 **마진 시뮬레이션** — 수수료/택배비/세금 반영 정밀 마진 계산
- 📊 **플랫폼 적합도** — 올웨이즈/토스 각각 몇 점인지 분석
- 🔗 **찾기→등록 원스톱** — 추천 상품 선택하면 바로 상품등록으로 연결

**기존 기능:**
- 상품명, 가격, 옵션명, 태그, 상세페이지 기획을 한 번에 생성

---

## 설치 방법 (3단계)

### 1단계: 압축 해제
다운받은 zip 파일의 압축을 풀어주세요.

### 2단계: 대시보드 실행
터미널을 열고 다음 명령어를 입력하세요:
```bash
cd 상품등록-서포터
bun run server.ts
```
→ 브라우저에서 **http://localhost:8765** 열기

### 3단계: Claude Code 실행 (다른 터미널)
```bash
cd 상품등록-서포터
claude
```

### 이렇게 사용하세요:
1. 브라우저 대시보드에서 엑셀 파일 드래그 → 자동 분석
2. 마음에 드는 상품 클릭 → "이 상품 선택" 버튼 클릭
3. 클로드 코드에서 "선택한 상품 등록해줘" 입력
4. → 상품명/가격/옵션/태그/상세페이지 자동 생성!

## 사용법

### 🔍 상품 찾기 (v2 신규!)

```
뭐 팔면 좋을까?
```
```
지금 시즌에 잘 팔리는 식품 추천해줘
```
```
공산품 도매매에서 여름 상품 찾아줘
```
→ 시즌/트렌드 분석 + 마진 시뮬레이션 + 플랫폼 적합도까지 한번에!

### 📦 상품 등록

#### 방법 1: 자연어로 말하기 (가장 쉬움)
```
사과 상품 등록하고 싶어
```
```
상호명 황금상회, 사과 5kg, 원가 8000원으로 상품 만들어줘
```
부족한 정보가 있으면 하나씩 물어봅니다.

#### 방법 2: 슬래시 커맨드
```
/상품등록
```

#### 방법 3: 엑셀 파일로 한꺼번에
```
이 엑셀 파일로 상품 등록해줘 /Users/내컴퓨터/상품목록.xlsx
```

#### 방법 4: 도매사이트에서 상품 찾기
```
도매사이트에서 사과 찾아줘
```
브라우저가 자동으로 도매사이트에 접속합니다. 로그인만 직접 해주세요.

### 🔗 찾기 → 등록 원스톱
```
상품 찾아줘 → (추천 결과 중) 1번 상품 등록해줘
```
→ 자동으로 상품 정보가 등록 스킬로 전달됩니다.

## 생성되는 결과물

| 항목 | 설명 |
|------|------|
| 올웨이즈 상품명 | 3개 안 (★별표 + 키워드 중복 방식) |
| 토스쇼핑 상품명 | 3개 안 (깔끔한 설명형) |
| 가격표 | 공격적/기본/널널 3단계 + 플랫폼별 세팅값 |
| 옵션명 | 황금코볼트식 (수식어 포함) |
| 태그 | 5~6개 (올웨이즈용) |
| 상세페이지 기획 | 간이/보통/집중 3단계 선택 |
