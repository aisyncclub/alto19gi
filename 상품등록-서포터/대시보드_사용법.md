# 대시보드 사용법

## 대시보드 켜는 방법

클로드 코드에서 아래처럼 말하면 됩니다:

```
마진계산기 켜줘
```

또는 이런 말도 됩니다:
```
대시보드 켜줘
대시보드 실행해줘
서버 켜줘
```

---

## 대시보드 접속 주소

서버가 켜지면 브라우저에서 아래 주소로 접속하세요:

```
http://localhost:8765/dashboard.html
```

또는 터미널에서 직접 열기:
```bash
open http://localhost:8765/dashboard.html
```

---

## 직접 서버 켜기 (클로드 없이)

터미널에서 아래 명령어를 입력하면 됩니다:

```bash
cd ~/dev_test1/alto19gi/상품등록-서포터
python3 -m http.server 8765
```

그 다음 브라우저에서 `http://localhost:8765/dashboard.html` 접속

---

## 서버 끄기

터미널에서 `Ctrl + C` 누르면 서버가 종료됩니다.

또는:
```bash
lsof -ti:8765 | xargs kill
```

---

## 대시보드 주요 기능

| 탭 | 기능 |
|----|------|
| 상품 찾기 | 도매 엑셀 업로드 → 시즌/마진 분석 → 추천 상품 선택 |
| 마진 계산기 | 원가/판매가 입력 → 전략별 마진 비교 + 인포그래픽 |
| 저장된 상품 | 마진 계산한 상품 저장/수정/삭제 관리 |

---

## 문제 해결

### "사이트에 연결할 수 없음" 오류
→ 서버가 꺼져있습니다. 위 방법으로 다시 켜주세요.

### "포트가 이미 사용 중" 오류
→ 이미 서버가 켜져있습니다. 브라우저에서 바로 접속하세요.
→ 그래도 안 되면: `lsof -ti:8765 | xargs kill` 후 다시 켜기
