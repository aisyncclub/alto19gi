# OSMU 멀티플랫폼 콘텐츠 가이드라인

## 공통 규칙 (모든 플랫폼 적용)

### 클리셰 금지 목록

AI가 자주 남용하는 진부한 표현은 절대 사용하지 않는다. 발견 시 자연스러운 대체 표현으로 교체할 것.

**금지 표현 목록:**
- 게임 체인저(Game Changer)
- 패러다임 시프트
- 혁신적인
- 획기적인
- 새로운 지평
- 판도를 바꾸다
- ~의 시대가 열리다
- 비밀 무기
- 놀라운 비법
- 당신의 인생을 바꿀
- 한 차원 높은
- 넥스트 레벨(Next Level)
- 차별화된
- 압도적인
- ~만 알면 된다
- 지금 바로 시작하세요
- 꿀팁 대방출
- ~하면 인생이 달라진다
- 단 하나의 비결
- 성공의 열쇠
- 모든 것을 바꿀
- 완벽한 솔루션
- 궁극의
- 마법 같은
- 폭발적인 성장

### 일관된 메시지 원칙
- 플랫폼 형식이 달라도 핵심 메시지는 프로필의 USP로 수렴
- 모든 콘텐츠가 결국 "이 서비스가 당신의 문제를 해결해 준다"로 연결
- 각 플랫폼에서 조금씩 다른 각도로 같은 핵심 메시지를 전달

### 데이터 기반 설득
- 구체적 수치 적극 활용 (매출, 시간 절약, 수강생 수 등)
- 막연한 표현("많은", "엄청난") 대신 정확한 숫자
- 비포/애프터 수치 비교가 가장 효과적
- 출처를 밝힐 수 있는 데이터는 출처 표기

### CTA 삽입 원칙
- 각 플랫폼 성격에 맞는 자연스러운 행동 유도
- 블로그/카페: 링크 직접 삽입
- 쓰레드: 댓글로 CTA
- 링크드인: 본문 말미에 자연스럽게
- 인스타: 캡션 + 프로필 링크 안내

### 프로필 금지어 체크
- 콘텐츠 생성 후 프로필에 등록된 금지어가 포함되었는지 반드시 검수
- 금지어 발견 시 대체 표현으로 자동 교체
- 교체 사실을 사용자에게 알림

## 1. 네이버 블로그

### 목적
상세한 정보 제공, 검색(SEO)을 통한 오가닉 트래픽 확보, 신뢰도 구축

### 형식
- 1,500자 이상 장문 아티클
- 소제목(##)으로 섹션 구분

### 구조
- **제목:** 검색 키워드 포함 + 호기심 유발. 예: "스마트스토어 상세페이지, AI로 10분 만에 끝내는 현실적인 방법"
- **도입부 (200~300자):** 타겟 고객의 페인 포인트 공감. "혹시 이런 경험 있으신가요?"
- **본문 (800~1,000자):** 구체적 방법론, 실제 사례(수강생 성과), 시각 자료 배치 위치 제안 [이미지: ○○ 캡처]
- **결론 + CTA (200~300자):** 핵심 정리 + 무료 리포트/웨비나 링크 유도

### 톤앤매너
전문적이고 친절한 강사/멘토. 진정성 강조. "~합니다" 체.

### SEO 규칙
- 핵심 키워드 제목 + 본문 3회 이상 자연스럽게 배치
- 관련 키워드 2~3개 추가 배치

## 2. 네이버 카페

### 목적
타겟 커뮤니티(셀러 카페, 맘카페 등) 내 자연스러운 바이럴 및 공감대 형성

### 형식
- 500~800자 경험담/팁 공유

### 구조
- **제목:** 커뮤니티 성격에 맞는 일상적 제목. 예: "요즘 상세페이지 만드느라 밤새시는 분들 계신가요? ㅠㅠ"
- **본문:** 스토리텔링 형식. "저도 예전엔 그랬는데..." → 전환점(AI 활용 발견) → 변화된 결과
- **결론:** 노골적 홍보 지양. 댓글 소통 유도 or 정보 공유 차원 가볍게 언급

### 톤앤매너
동료 셀러/이웃 느낌. 친근하고 솔직. "~요" 체. 이모티콘 적절히 사용.

### 주의사항
- 광고 티 나면 역효과. 자연스러운 경험 공유가 핵심
- "혹시 관심 있으시면 쪽지 주세요" 식의 소극적 CTA

## 3. 쓰레드 (Threads)

### 목적
짧고 강렬한 인사이트 제공, 빠른 확산 및 트래픽 유도

### 형식
- 4줄 구조
- **각 줄 12자 제한 (필수 준수)**

### 구조 (필수)
- **1줄:** 후킹 — 대중이 공감할 특정 키워드로 시작. 예: "상세페이지 외주비"
- **2줄:** 인사이트 1
- **3줄:** 인사이트 2
- **4줄:** 요약/의견

### 댓글 활용
- 핵심은 본문 4줄에 요약
- 상세 내용, 맥락, CTA 링크는 **댓글(reply)**로 제공

### 톤앤매너
'최신 업데이트 고급팁' 느낌. 전문성 유지 + 트렌디 + 간결.

### 제약사항 체크리스트
- [ ] 각 줄 12자 이하인가?
- [ ] 4줄인가?
- [ ] 1줄에 후킹 키워드가 있는가?
- [ ] CTA는 댓글에 있는가?

## 4. 링크드인 (LinkedIn)

### 목적
비즈니스 프로페셔널 대상 전문성 어필, B2B 네트워킹, 신뢰도 구축

### 형식
- 300~500자

### 구조
- **첫 2줄:** 핵심 인사이트로 시작 (접힘 전에 보이는 부분)
- **본문:** AI 도입 효율성, 이커머스 트렌드, 비즈니스 시스템화 등 거시적 관점
- **말미:** 자연스러운 CTA (의견 요청 or 리소스 안내)

### 필수 제약 (반드시 준수)
- **이모지(Emoji) 사용 금지**
- **볼드체(**Bold**) 사용 금지**
- 강조는 문장 구조, 줄 바꿈, 텍스트 기반

### 톤앤매너
논리적, 객관적, 프로페셔널. "~합니다" 체. 데이터와 논리로 설득.

### 제약사항 체크리스트
- [ ] 이모지 없는가?
- [ ] 볼드체 없는가?
- [ ] 300~500자인가?
- [ ] 비즈니스 인사이트 관점인가?

## 5. 인스타그램 (Instagram)

### 목적
시각적 콘텐츠로 브랜드 인지도 제고, 트렌드 민감층 공략

### 형식
- 캡션 + 카드뉴스 (3장 이상)

### 캡션 구조
- 첫 줄: 후킹 (피드에서 보이는 부분)
- 본문: 핵심 설명 (150~300자)
- 해시태그: 10~15개 (대중 + 니치 혼합)

### 카드뉴스 스타일 (필수)
- **아티스틱 미니멀 모노톤**: 흰 배경, 블랙/그레이 타이포그래피 중심, 에디토리얼 감성
- 인스타그램에 바로 올릴 수 있는 크기(1080x1080 또는 1080x1350)로 꽉 채움
- **박스 안의 박스 형태 지양**

### 카드뉴스 구성 (3~5장)
- 1장: 제목 (대형 타이포, 호기심 유발)
- 2~3장: 핵심 내용 (한 장에 한 메시지)
- 마지막 장: CTA (저장/공유 유도 or 프로필 링크 안내)

### 카드뉴스 제작 방식 (사용자 선택)
사용자에게 물어볼 것:
1. **HTML로 직접 만들기** — HTML/CSS로 카드뉴스 렌더링
2. **이미지 프롬프트(나노바나나)로 만들기** — 이미지 생성 AI용 프롬프트 제공

### 톤앤매너
세련되고 감각적이며 트렌디. 짧고 임팩트 있는 문장.

## 자체 검토 체크리스트 (콘텐츠 생성 후 필수)

생성된 모든 콘텐츠에 대해 다음을 검수:

1. [ ] 클리셰 금지 목록의 표현이 포함되지 않았는가?
2. [ ] 프로필의 금지어가 사용되지 않았는가?
3. [ ] 각 플랫폼의 글자수/줄수 제한을 준수했는가?
4. [ ] 쓰레드: 4줄, 각 12자 이하
5. [ ] 링크드인: 이모지/볼드 없음, 300~500자
6. [ ] 핵심 메시지가 프로필의 USP와 일관되는가?
7. [ ] CTA가 각 플랫폼에 적절하게 포함되었는가?
8. [ ] 데이터/수치가 포함되었는가?
