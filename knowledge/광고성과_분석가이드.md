# 광고 성과 분석 가이드

## 개요

올웨이즈/토스쇼핑 광고 성과 데이터를 분석하여 **시각적 HTML 리포트**와 **즉시 실행 액션 플랜**을 생성한다.

---

## 데이터 입력 형식

### 올웨이즈 광고 성과 엑셀 컬럼 (표준)

| 컬럼 | 설명 | 핵심 여부 |
|------|------|----------|
| 일자 | 날짜 | O |
| 캠페인 ID / 캠페인 | 캠페인 식별 | O |
| 광고 세트 ID / 광고 세트 | 광고 그룹 | O |
| 광고 ID / 광고 | 개별 광고 | O |
| 상품 ID / 상품 | 상품 정보 | O |
| 옵션 ID / 옵션 | 옵션 정보 | O |
| 노출수 | 광고 노출 횟수 | O |
| 클릭수 | 광고 클릭 횟수 | O |
| 클릭률 | CTR (%) | O |
| 전환율 | 구매 전환율 (%) | O |
| 총 전환 판매수량 | 총 판매 수량 | O |
| 직접 전환 판매수량 | 직접 전환 수량 | |
| 간접 전환 판매수량 | 간접 전환 수량 | |
| 총 전환 주문건수 | 총 주문 건수 | O |
| 총 전환 거래액 | 총 매출 | O |
| 직접/간접 전환 거래액 | 세분화 매출 | |
| 총 전환 광고수익률 | ROAS (%) | O |
| 집행 광고비 | 사용 광고비 | O |
| 클릭당 비용 | CPC | O |

### 데이터가 없는 컬럼은 무시하고, 있는 데이터로 최대한 분석한다.

## 분석 프로세스

### 1단계: 데이터 집계

```
[캠페인별 집계]
- 캠페인명, 상품명
- 총 노출, 총 클릭, 평균 CTR
- 총 전환, 총 매출, ROAS
- 총 광고비, 평균 CPC

[옵션별 집계]
- 옵션명 (캠페인 구분 포함)
- 노출, 클릭, CTR, 전환, 매출, 광고비, CPC
- CTR 순위, CPC 순위 매기기

[일별 집계]
- 날짜별 총 노출, 클릭, 전환, 광고비
- 추이 패턴 파악 (상승/하락/급변)
```

### 2단계: KPI 산출 & 벤치마크 비교

| KPI | 산출법 | 업계 평균 (올웨이즈 기준) |
|-----|--------|-------------------------|
| CTR | 클릭수 ÷ 노출수 × 100 | 1~3% (양호), <0.5% (위험) |
| CVR | 전환수 ÷ 클릭수 × 100 | 3~10% (양호), 0% (심각) |
| CPC | 광고비 ÷ 클릭수 | 10~50원 (양호), >100원 (비효율) |
| ROAS | 매출 ÷ 광고비 × 100 | 300%+ (양호), <200% (적자) |
| 일 광고비 | 총 광고비 ÷ 기간 일수 | 최소 3,000~5,000원/일 권장 |

### 3단계: 문제 진단

아래 체크리스트를 순서대로 확인한다:

```
[퍼널 분석] — 어디서 이탈하는가?
□ 노출 → 클릭 이탈률 (CTR 문제) → 썸네일/상품명/가격 문제
□ 클릭 → 전환 이탈률 (CVR 문제) → 상세페이지/가격/리뷰 문제
□ 전환 0건 → 둘 다 문제 or 가격 경쟁력 없음

[시즌 적합성]
□ 상품이 현재 시즌에 맞는가? (예: 4월에 부사사과 = 비수기)
□ 제철 상품인가? 비수기 상품인가?

[구조 효율성]
□ 같은 상품이 여러 캠페인에서 중복 운영되고 있지 않은가?
□ 옵션이 너무 많이 분산되어 있지 않은가?
□ 캠페인명/광고세트명이 의미 있게 설정되어 있는가?

[예산 적정성]
□ 일 광고비가 유의미한 데이터를 모을 수 있는 수준인가?
□ 성과 좋은 옵션에 예산이 집중되어 있는가?

[CPC 효율성]
□ CPC가 비정상적으로 높은 옵션이 있는가? (>100원)
□ 낮은 CTR → 높은 CPC 패턴인가?
```

### 4단계: 개선안 도출

문제 심각도별 분류:

```
[Critical — 즉시 조치]
- 전환율 0% 지속
- 비수기 상품 광고 집행
- CPC 200원 이상

[Warning — 빠른 개선]
- CTR < 업계 평균
- 캠페인 중복 운영
- 예산 과도한 분산

[Opportunity — 전략 제안]
- 성과 좋은 옵션 집중 투자
- 타임특가 활용
- 광고비 증액 제안
- 긍정 시그널 활용
```

### 5단계: 액션 플랜

반드시 **우선순위 5개** 이내의 즉시 실행 가능한 액션으로 정리:
1. 가장 급한 것 (중지/제거)
2. 구조 개선 (통합/정리)
3. 집중 투자 (베스트 옵션)
4. 콘텐츠 개선 (상세페이지/썸네일)
5. 프로모션 활용 (타임특가/할인)

## HTML 리포트 생성 규칙

### 필수 포함 섹션

1. **경고 배너** — 가장 심각한 문제를 상단에 크게 표시
2. **KPI 카드** — 총 노출, 클릭, CTR, 전환/매출, 광고비 (5개)
3. **전환 퍼널** — 노출 → 클릭 → 구매 단계별 이탈률
4. **일별 추이 차트** — 노출수 & 클릭수 바 차트
5. **캠페인별 비교** — 카드형 사이드바이사이드 비교
6. **옵션별 상세** — 바 차트 + 테이블 (CTR/CPC 컬러코딩)
7. **CPC 비교** — 옵션별 CPC 바 차트 (낮은→높은 순)
8. **개선 영역** — Critical/Warning/Opportunity 카드
9. **즉시 실행 액션** — 번호 매긴 실행 계획
10. **현재 vs 개선 후 예상** — 비교 테이블

### 디자인 규칙

- **다크 테마** 기본 (배경 #0a0a0f)
- **컬러 코딩**: 빨강=위험/심각, 주황=경고, 초록=양호/기회, 파랑=정보/노출
- **폰트**: Noto Sans KR (본문) + JetBrains Mono (숫자) + Playfair Display (제목)
- **애니메이션**: fadeIn, barGrow 등 진입 애니메이션
- **반응형**: 모바일에서도 읽을 수 있게
- `templates/ad_report_template.html`을 기반으로 실제 값을 치환하여 `data/ad_report.html` 생성

### 컬러코딩 기준

| 지표 | 위험 (빨강) | 경고 (주황) | 양호 (초록) |
|------|------------|------------|------------|
| CTR | <0.3% | 0.3~1.0% | >1.0% |
| CVR | 0% | <3% | >3% |
| CPC | >100원 | 50~100원 | <50원 |
| ROAS | 0% | <300% | >300% |
| 마진율 | <15% | 15~25% | >25% |

## 터미널 출력 형식

HTML 리포트와 별도로, 터미널에 **핵심 요약**을 먼저 출력한다:

```
━━━ 광고 성과 분석 ━━━

기간: YYYY.MM.DD ~ YYYY.MM.DD (N일)
캠페인: N개 / 광고: N개 / 옵션: N개

📊 핵심 KPI
  노출: OO,OOO / 클릭: OOO / CTR: O.OO%
  전환: O건 / 매출: O원 / ROAS: O%
  광고비: O,OOO원 (일 평균 OOO원)

⚠️ 가장 심각한 문제
  (1줄 요약)

✅ 즉시 해야 할 것
  1. ...
  2. ...
  3. ...

📄 상세 리포트: data/ad_report.html
```

## 특수 상황 처리

### 전환 0건인 경우
- 반드시 "상세페이지 + 가격 + 리뷰" 3가지를 점검하라고 안내
- 타임특가 활용한 첫 전환 만들기 전략 필수 포함

### 데이터가 적은 경우 (3일 미만 또는 노출 1,000 미만)
- "데이터가 부족하여 정확한 분석이 어렵습니다" 면책 추가
- 최소 7일 이상 운영 후 재분석 권장

### 비수기 상품 감지
- 과일 시즌 참조: 봄(딸기/참외), 여름(수박/복숭아), 가을(사과/배/감), 겨울(귤/한라봉)
- 비수기 상품 광고 시 반드시 경고 + 대안 제시

### 여러 플랫폼 데이터
- 올웨이즈와 토스쇼핑 데이터가 섞여 있으면 플랫폼별로 분리 분석
- 각 플랫폼 수수료 구조 차이를 반영하여 실제 마진까지 연계 분석
